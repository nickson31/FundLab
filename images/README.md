# FundLab Logo - Pure Wordmark

## Design Philosophy
**Minimal. Professional. Typographic.**

No symbols. No complex colors. Just clean typography.

---

## Files

### fundlab-logo.svg (Primary)
- Pure wordmark: "FundLab"
- Color: Dark gray #111827
- Font: Inter Bold 700
- Use: Website, documents, presentations

### fundlab-icon.svg (Favicon)
- Single letter: "F"
- White on dark square
- Use: Browser favicon, app icon

### fundlab-logo-white.svg (Dark Mode)
- White wordmark
- Use: Dark backgrounds

---

## Inspiration
- **Stripe**: Pure wordmark, no symbol
- **Netflix**: Just text, strong typography
- **Figma**: Wordmark-focused
- **Linear**: Clean typography

---

## Usage
```html
<!-- Header -->
<img src="/images/fundlab-logo.svg" alt="FundLab" height="32" />

<!-- Favicon -->
<link rel="icon" href="/images/fundlab-icon.svg" />
```
