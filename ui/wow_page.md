## Key Copywriting Principles

1. **Benefit > Feature**: "Save 50 hours of research" not "AI algorithm"
2. **Specificity**: "500+ investors" not "many investors"
3. **Urgency**: "Start today" not "eventually"
4. **Social Proof**: Numbers, testimonials, trust badges
5. **Clarity**: Avoid jargonâ€”speak like a human

## Final Notes for Developers

- Use **next/image** for optimized image loading
- Implement **lazy loading** for sections below fold
- Add **meta tags** for SEO (title, description, OG image)
- Include **Google Analytics** or Plausible
- Ensure **accessibility**: alt text, keyboard navigation, ARIA labels
